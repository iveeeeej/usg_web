# Campus Connect
## University of Student e-Governance and Services

## 1. Project Description

Campus Connect is a web and mobile-based e-governance platform designed to centralize and automate administrative, communication, and service processes of the University Student Government (USG).

The system replaces fragmented, manual processes with a structured digital workflow system.

---

## 2. Core Objectives

- Centralize communication and announcements
- Automate attendance and violation tracking
- Provide structured reporting templates
- Enable digital service workflows (borrow, payment, lost & found)
- Improve accountability through data logging
- Provide analytics for decision-making
- Integrate web admin + mobile student interface

---

## 3. User Roles

### 1. Super Admin
- Full system control
- Manage organizations
- Manage officers
- Manage system-wide configurations

### 2. Organization Admin / Officers
- Manage events
- Approve borrow requests
- Publish announcements
- Manage reports
- Monitor payments

### 3. Students
- View events and announcements
- Scan QR for attendance
- Submit borrow requests
- View violations and payments
- Submit feedback
